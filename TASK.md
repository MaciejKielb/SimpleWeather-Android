Build an Android weather application with the following features:
Weather List Screen
- Shows a list of 5 cities current weather 
- Shows current weather for user’s location at the top of the list
City Detail Screen
When a city in the list is tapped, it navigates to a detail screen.
Detail screen shows extended weather information (e.g., temperature, forecast, humidity, wind, etc.).

The design is free form — you can use any UI/UX style.
